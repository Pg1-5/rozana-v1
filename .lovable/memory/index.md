# Project Memory

## Core
Dark UI #0e0e0e, lime green #c8f060 primary, amber #f0c060 secondary, cyan #60d4f0 tertiary.
Playfair Display headings, DM Sans body. Never Inter/Roboto.
No red warnings, no failure language, no streaks — momentum only.
Mobile-first max-w-520px centered. Indian health audience.
App name is Rozana (not Vitale). Use #Rozana in hashtags and branding.
